💬 Portfolio Chatbot

Un portfolio qui répond à vos questions — propulsé par l'IA.


🧠 L'idée

La plupart des portfolios sont des pages statiques : une liste de projets, un CV mis en forme, un formulaire de contact.
Celui-ci fonctionne différemment.
Portfolio Chatbot est une interface conversationnelle qui vous permet de poser directement vos questions sur mon parcours, mes compétences, mes projets ou mes disponibilités. Une IA vous répond en temps réel, à partir d'un contexte que j'ai entièrement rédigé moi-même.
Pas de navigation. Pas de scroll. Juste une question, et une réponse.

✨ Pourquoi ce format ?

En tant que développeur en formation (Bachelor Concepteur et Développeur d'Applications), je voulais que mon portfolio reflète ce que j'aime dans le développement : construire des choses utiles, avec une vraie intention derrière.
Un portfolio classique contraint le visiteur à chercher l'information qu'il veut. Ici, il la demande — et l'obtient directement.
C'est aussi une façon d'explorer concrètement l'intégration d'une API IA dans une application web, de bout en bout : front-end, back-end, sécurité, déploiement.

⚙️ Outils utilisés

HTML / CSS / JavaScript, Node.js, Express, oogle Gemini, GitHub Pages, Render

🔐 Sécurité de la clé API

La clé API n'est jamais exposée côté client. Toutes les requêtes vers Gemini transitent par un mini-serveur Node.js, invisible au navigateur. Le front-end ne communique qu'avec ce serveur intermédiaire.
