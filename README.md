💬 Mon Portfolio Chatbot : Quent2.0

Un portfolio qui répond à vos questions, propulsé par l'IA.


🧠 L'idée

La plupart des portfolios sont des pages statiques : une liste de projets, un CV mis en forme, un formulaire de contact.
Je souhaitais que le mien fonctionne différemment.
Quent2.0 est une interface conversationnelle qui vous permet de poser directement vos questions sur mon parcours, mes compétences, mes projets, ou tout autre information dont un rucruteur peux avoir besoin. Une IA vous répond en temps réel, à partir d'un contexte que j'ai entièrement rédigé moi-même.
Pas de navigation. Pas de scroll. Juste une question, et une réponse.

✨ Pourquoi ce format ?

En tant que futur alternant au Cesi a la  Bachelor Concepteur et Développeur d'Applications, je voulais que mon portfolio reflète ce que j'aime dans le développement : construire des choses utiles, pratique et qualitative, avec une vraie intention derrière.
Hier, nous devions aller chercher les information que l'on desirais par nous meme, comme dans un portfolio classique.
Aujourd'hui, les technologies nous permettent de plus en plus de simplement demander une information et de l'obtenir directement.
C'est aussi une façon d'explorer concrètement l'intégration d'une API IA dans une application web, de bout en bout : front-end, back-end, sécurité, déploiement. Technique, qui pour certaine, m'etais totalement inconnue. J'etais ravi de decouvrir toutes ces nouvelles chose en construsant ce portfolio. 

⚙️ Outils utilisés

HTML / CSS / JavaScript, Node.js, Express, oogle Gemini, GitHub Pages, Render

🔐 Sécurité de la clé API

La clé API n'est jamais exposée côté client. Toutes les requêtes vers Gemini transitent par un mini-serveur Node.js, invisible au navigateur. Le front-end ne communique qu'avec ce serveur intermédiaire.
